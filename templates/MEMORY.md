# MEMORY.md — Long-Term Memory

Curated, durable context the agent should know. Loaded at session start in private/direct contexts only.

Keep this lean — under ~10,000 characters where possible. Semantic search works better on focused files. Promote daily-log entries here when they become durable; archive old entries into `memory/archive/` when they go stale.

---

## 🚨 Iron Laws

*House rules set by the human. Non-negotiable. Add new entries dated, never edit past ones — supersede them with a new one if needed.*

Example format:

```
## 🚨 IRON LAW: [Rule name] (set YYYY-MM-DD)
[The rule in one or two sentences.]
[Why it matters.]
[What NOT to do.]
```
